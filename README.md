# 📦 Final Project — Sanbercode Next.js

Proyek akhir dari program bootcamp **Sanbercode** yang berfokus pada pengembangan aplikasi web menggunakan **Next.js**.

## 🛠️ Tech Stack

- **Next.js** — React framework
- **JavaScript** — Bahasa utama

## 🚀 Cara Menjalankan

```bash
npm install
npm run dev
```
